Personal portfolio of Abhay Pandey, AI Products Lead at SapienOne.

## 🚀 Live
Deployed via GitHub Pages from `main` (auto-deploys on push).

## ✨ What's on the page
- **Hero + dual CTA** — email capture for the free AI Leadership Team, plus a jump to shipped products
- **Currently / Previously** — SapienOne now; FuturePath AI, HCL, Ikigai before
- **Building now @ SapienOne** — LEAP, Cur8, SPINE
- **Products shipped** — Albusnote, IVR Labs, Ideaship, BharatLearn & CivicSupport
- **Free value** — AI Leadership Team + PM Advisor & Coach (email capture → Google Sheets → GitHub/Drive)
- **About, Experience, Contact**

## 🛠️ Tech
- Static HTML / CSS (custom properties) / vanilla JS — no build step
- Inter (Google Fonts)
- Email capture posts to a Google Apps Script endpoint (`google-apps-script.js`)

## 📁 Structure
